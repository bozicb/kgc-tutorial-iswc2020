# How to run Morph-CSV?

```bash
cd data
bash download.sh #to download BSBM and Bio2RDF tabular sources
cd ..
docker-compose up -d
```
And an user interface will be deployed at localhost:5000. More info at https://github.com/oeg-upm/morph-csv
